# BFB Mixins

A Mixin mod for **BFB - Brutal modpack** (Minecraft 1.20.1 Forge).


## Description

BFB Mixins is a personal balancing and tweaking mod designed for the
**BFB - Brutal modpack**. It uses Mixins to modify and rebalance various
mods in the modpack without editing their source code directly, making
it easy to update and maintain.

This mod aims to improve the overall gameplay experience by adjusting
mod interactions, rebalancing overpowered or underpowered mechanics,
and making different mods work better together in the modpack environment.

> ⚠️ **Note**: This mod is specifically designed for the BFB modpack.
> Using it outside of the intended modpack may cause unexpected behavior.
